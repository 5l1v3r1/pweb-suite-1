SMSI (SimplyMySQLi)

_**About**_

MySQL SQL Injection Vulnerability finding tool.

_**Depends**_

use LWP::UserAgent (Perl)
use Term::ANSIColor (Perl)

_**Manual**_

Usage: ./smsi -u '< URL >' -a < User Agent >
> Put URL in single quotes to escape special characters from the shell.
